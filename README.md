# AICTE-Internship-potato-leaf-detection
🌿 Plant Disease Detection System
An AI-powered web application designed to help farmers identify diseases in potato leaves instantly. Using Deep Learning, the tool provides a diagnosis, confidence score, and actionable treatment solutions to protect crop yields.

🔗 https://agroscanweb.streamlit.app/

🚀 Overview
The system simplifies crop monitoring by allowing users to upload leaf images and receive immediate feedback on plant health.

Instant Recognition: Detects Healthy, Early Blight, and Late Blight in potato plants.

Expert Solutions: Provides specific fungicide and maintenance recommendations for each diagnosis.

History Tracking: Automatically keeps a log of all previous predictions for future reference.

Global Insights: Contextual facts about the economic and historical impact of specific plant diseases.

🛠️ Tech Stack
AI Engine: TensorFlow & Keras (CNN Model)

Web Interface: Streamlit

Data Handling: NumPy & Pandas

Image Processing: Pillow (PIL)

💻 How to Run Locally
Clone the repository and navigate into the folder.

Install the requirements: pip install -r requirements.txt.

Ensure your .keras model file is in the folder.

Launch the app: streamlit run app.py.
